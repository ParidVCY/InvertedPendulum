# InvertedPendulum
Inverted Pendulum by **Parid Vetchaiyo**\
Using PID (Proportional-Integral-Derivative) and LQR (Linear Quadratic Regulator) controllers and implemented on the ESP32.
# Hardware List
* 20x20 Gantry Plate with 3 wheels
* Timing Belt
* Pulley 20 teeth
* Shaft 8 mm diameter, 22 cm length
* NEMA17 stepper motor
* TB6600 Microstep Driver
* Encoder bore size 8 mm
* Power Supply 24V
